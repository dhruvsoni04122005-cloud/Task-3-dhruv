🔐 Task 03: Random Password Generator

📌 Description
This program generates a strong random password using:
A user-defined keyword
Birthdate
Random characters (letters, numbers, symbols)

⚙️ Features
Custom password length (8–20 characters)
Mix of uppercase, lowercase, digits, and special characters
Shuffled output for better security

▶️ How to Run
python task_03.py
🧾 Example
Enter keyword → abc
Enter birthdate → 01012000
Length → 10
Output → aB@01cD!2X
